# Mabel-Backend

Versão 2.0 do Sistema Mabel, com foco maior em **Orientação a Objetos**,   
**Testes** e a adição de **Eventos** (Event-Driven Architecture). Entidades    
do banco de dados repensadas para se encaixar ao sistema físico da escola,   
serem resistentes à mudanças e com histórico de dados valioso. Sistema   
de pagamentos refinado e mais abrangente, com possibilidade de integração   
a sistemas externos.

## Stack

- Typescript
- Node.js
- Express.js
- Vitest
- Docker
- JWT
- bcrypt
